# 大千分會 · 攻綠燈儀表板 🚦

BNI 大千分會的「攻綠燈」計分儀表板。上傳 PALMS 成績報告，自動算每位夥伴六項分數與紅黃綠燈，並列出到月底上 70 分綠燈還要補多少。本期週數依 PALMS 出席資料**自動推算**，不需手動輸入。

## 線上網址（GitHub Pages）

啟用 Pages 後：<https://geniusjohn0502.github.io/greenlight-dashboard/>

打開即看，夥伴只需這個連結。

## 每週更新（管理者）

1. 到本 repo 的 [`data/`](data/) 資料夾
2. 點 **Add file → Upload files**，把本週的 PALMS 檔（`.xls` / `.csv`）拖上去
3. **檔名建議含日期**（例如 `2026-07-13.xls`）—— 網頁會自動抓「檔名最新」那一份
4. **Commit changes**
5. 夥伴重新整理網頁 → 看到最新資料、週數自動重算

> 網頁開啟時會自動讀取 `data/` 裡最新一份 PALMS 檔（透過 GitHub API + raw）。舊檔可留著當歷史，或刪除只保留最新。

## 計分規則（近 6 個月滾動；總分＝六項相加，滿分 100）

- 燈號：綠 ≥70、黃 50–69、紅 <50、灰＝新夥伴（點列內可標記，不列統計）
- **固定級距**　出席：0缺20 / 1缺15 / 2缺10 / 3缺↑0　·　培訓學分：≥6→15 / ≥4→10 / ≥2→5　·　引薦金額：≥200萬→15 / ≥80萬→10 / ≥40萬→5
- **依本期週數換算**　引薦單(內+外,每週)：≥1.5→20 / ≥1.2→15 / ≥1.0→10 / ≥0.75→5　·　一對一(每週)：≥2→15 / ≥1→10 / ≥0.5→5　·　來賓(每四週)：≥2→15 / ≥1→10
- **本期週數自動推算**＝全分會「出席+缺席+遲到+病假+替代」最高者（資歷最完整夥伴經歷的例會數），已自動排除停會週；可在頁面手動覆寫
